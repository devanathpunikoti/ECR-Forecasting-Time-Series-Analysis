# ECR Revenue Forecasting using Time Series Analysis

## Project Overview

This project focuses on forecasting ECR revenue using Time Series Analysis and Machine Learning techniques.

The objective is to identify trends, seasonality, and revenue patterns to generate accurate forecasts and support business decision-making.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Statsmodels

---

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Seasonal Decomposition
5. Feature Engineering
6. Model Training
7. Performance Evaluation
8. Forecast Visualization

---

## Machine Learning Models

- Decision Tree Regressor
- Random Forest Regressor
- Gradient Boosting Regressor

---

## Evaluation Metrics

- MAE (Mean Absolute Error)
- R² Score

---
## Model Performance

| Model | MAE | R² Score |
|---------|---------|---------|
| Decision Tree | 4,336,942 | 0.513 |
| Random Forest | 5,068,861 | 0.371 |
| Gradient Boosting | 3,586,185 | 0.662 |

Best Model: Gradient Boosting

** MAPE: 7.62% **

---

## Key Outcomes

- Identified trend and seasonal patterns.
- Built forecasting models using engineered time-series features.
- Compared multiple machine learning approaches.
- Generated revenue forecasts for business planning.
